# IT-Ticket-Analysis-Dashboard Requirements
IT Ticket Analysis Dashboard

**Project Overview**
This project analyzes IT service tickets to track performance, SLA compliance, and customer satisfaction.

**Objectives**
- Monitor ticket volume
- Track SLA compliance
- Analyze resolution time
- Improve customer satisfaction

**Key Metrics**
- Total Tickets: 97K
- SLA Compliant: 48%
- SLA Non-Compliant: 52%
- Avg Resolution Time: 4.55
- Avg Satisfaction: 4.10

**Dashboard Features**
- Tickets by Severity
- Tickets by Priority
- Tickets by Issue Type
- SLA Performance Tracking
- Yearly & Quarterly Trends

**Tools Used**
- Microsoft Power BI
- Excel / CSV Dataset
